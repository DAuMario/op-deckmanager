# One Piece TCG Deck Manager

Ein Tool zum Verwalten und Analysieren von One Piece TCG Decks, mit KI-gestützten Vorschlägen zum Deckbau. 

## Status

In Entwicklung

## Features (geplant)

- Karten und Decks verwalten
- Statistische Deck Analyse (Kostenkurve, Farbverteilung, Counter)
- KI-gestützte Deckbau-Tipps über eine LLM-API
